# third-party-js-dev

> Simulating multiple domains at local for third-party JavaScript development

## Prerequisite

Add the following entries to your hosts file:

```
127.0.0.1       publisher.dev
127.0.0.1       widget.dev
```

## Usage

### Setup

```
$ npm install
```

### Running

```
$ npm start
```

### Testing

```
$ npm test
```
